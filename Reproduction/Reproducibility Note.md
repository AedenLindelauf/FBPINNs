# Reproducibility note
To obtain the figures used in the report, one needs to run the appropriate code and open the results in tensorboard. 
These results can then be exported into a csv file, which can be imported in the jupyter notebook plots_reproduction.ipynb.



To reproduce the figures in section 4, run the code in `Stress testing - 1D u'' = f.ipynb` and then run the command `tensorboard --logdir results/summaries/`.
